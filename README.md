# pbo
tugasakhir


### 1. Layer MODEL (Dikerjakan oleh: asa)
* **Fokus Direktori:** `src/main/java/model/`
* **Tugas:** Membuat struktur data, relasi, dan perhitungan.

* **File yang dikelola:** `Putusan.java`, `KnowledgeRepository.java`, `StatistikPutusan.java`.

### 2. Layer VIEW (Dikerjakan oleh: amil)
* **Fokus Direktori:** `src/main/java/view/` & `src/main/resources/com/example/pbo/`
* **Tugas:** Membuat tampilan antarmuka (Console atau JavaFX) dan menerima input *user*.
* **File yang dikelola:** `ConsoleView.java`, `JavaFXView.fxml`, (dan file CSS jika ada).

### 3. Layer CONTROLLER (Dikerjakan oleh: Octa)
* **Fokus Direktori:** `src/main/java/controller/`, `src/main/java/util/`, `src/main/java/app/`
* **Tugas:** Menjadi jembatan antara Model dan View, validasi input (try-catch), dan mengatur alur program di `Main.java`.
* **File yang dikelola:** `KnowledgeController.java`, `InputHandler.java`, `Main.java`.
